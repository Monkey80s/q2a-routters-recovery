# Routters Recovery Tool para Question2Answer

Este plugin permite gestionar el CSS de seguridad y el control de caché directamente desde el panel de administración de Q2A.

## Características
- **Master CSS:** Inyecta código de respaldo que sobreescribe errores del tema.
- **Cache Buster:** Cambia la versión del CSS para forzar la actualización en navegadores móviles.
- **Interfaz Intuitiva:** Edición directa desde la administración.

## Instalación
1. Descarga el repositorio.
2. Sube la carpeta a `qa-plugin/`.
3. Activa y configura en **Administración > Plugins > Routters Recovery**.
